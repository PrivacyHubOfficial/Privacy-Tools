# Version 3 - Verification and Reusability

## Objective

Transform the project into a reusable and more reliable privacy tool.

This version focuses on software design and validation.

---

## Components

### Reader

The reader is implemented using reusable functions that return structured metadata.

This makes the code easier to integrate into larger applications.

### Remover

The remover reconstructs the image, exports a cleaned version, and verifies that metadata has been removed successfully.

Example:

Clean removal confirmed: No metadata found.
